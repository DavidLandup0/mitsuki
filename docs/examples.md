# Application Examples

This page provides an overview of the example applications available in the project. These examples demonstrate various features of the Mitsuki framework.

## Blog App

The `blog_app` example is a simple blogging application that demonstrates core concepts such as controllers, services, repositories, and more.

[View on GitHub](https://github.com/DavidLandup0/mitsuki/tree/main/examples/blog_app)

## OpenAPI

The `openapi` example shows how to generate OpenAPI documentation for your application. This is useful for creating interactive API documentation and for sharing your API specification with other developers.

[View on GitHub](https://github.com/DavidLandup0/mitsuki/tree/main/examples/openapi)

## Scheduled Tasks

The `scheduled` example demonstrates how to run tasks at a specified interval. This is useful for background jobs, such as sending emails or cleaning up old data.

[View on GitHub](https://github.com/DavidLandup0/mitsuki/tree/main/examples/scheduled)
